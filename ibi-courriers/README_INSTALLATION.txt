================================================================
IBI COURRIERS - Guide d'installation
================================================================

1. Copier le dossier dist\IBI_COURRIERS\ sur le PC cible
   (contient IBI_COURRIERS.exe et le sous-dossier _internal\)

2. Lancer IBI_COURRIERS.exe

3. Au premier lancement, l'application cree automatiquement :
   courriers.db, uploads\, exports\, backups\ a cote de l'exe

4. Connexion par defaut (premiere installation uniquement) :
   E-mail       : admin@ibi.local
   Mot de passe : admin123

   IMPORTANT : Changer le mot de passe administrateur des la premiere
   connexion (menu Utilisateurs, reserve admin).

Sauvegarde :
   Dossier backups\ (sauvegarde automatique quotidienne a minuit
   si l'application tourne)
   Gestion manuelle : menu Sauvegardes (admin et DG)

Support : alexandreassignon@gmail.com
